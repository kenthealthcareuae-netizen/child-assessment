# Child Assessment EN - English Landing Page

This is the English version of the Kent Healthcare child assessment landing page, focused on specialized child assessment and development services.

## Features

- **Neurological Physiotherapy**: Advanced programs for stroke rehabilitation, Parkinson's disease, and neurological disorders
- **Customized Recovery Plans**: Personalized treatment strategies for physical and motor rehabilitation
- **Clinic & Home Sessions**: Flexible treatment options in Dubai
- **Elderly & Family Support**: Specialized programs for elderly care and family training
- **Comprehensive Assessments**: In-depth evaluations with continuous monitoring
- **Quality & Trust**: Licensed therapists with proven protocols

## Services Covered

- Post-stroke rehabilitation and cognitive re-learning
- Parkinson's disease management
- Neurological disorders (Multiple Sclerosis, brain injuries)
- Musculoskeletal injuries with behavioral impact
- Movement problems related to fear and anxiety
- Post-surgical treatment (joint replacement, fractures)

## Technology Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI Components
- EmailJS integration
- Google Analytics & Meta Pixel tracking
